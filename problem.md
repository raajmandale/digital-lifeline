# The Data Loss Problem

Modern digital life depends on data.

Photos, research, code, financial records, and business documents all rely on storage systems.

Yet data loss remains extremely common.

## Common Causes

- ransomware attacks
- SSD or HDD failure
- accidental deletion
- corrupted storage
- cloud account suspension
- stolen devices

Many users rely on backups, but backups introduce another assumption:

> The backup location will always exist.

If the backup itself fails, data may still be lost.

## The Core Question

How can we design systems where **no single failure can destroy data?**

Digital Lifeline explores a survivability architecture designed around that principle.
